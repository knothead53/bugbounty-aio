# BugBounty-AIO

Scaffold for a self-hosted bug bounty orchestration toolkit.

This repo will evolve in small phases:
- backend (FastAPI orchestrator)
- frontend (React UI)
- docker-compose
- docs & setup guides

## Repo Structure

├── backend/ # FastAPI orchestrator
│ ├── main.py
│ ├── init.py
│ └── requirements.txt
├── frontend/ # React UI
│ └── README.md
├── compose/ # Docker orchestration
│ └── docker-compose.yml
├── docs/ # Documentation & guides
├── .gitignore
└── README.md
